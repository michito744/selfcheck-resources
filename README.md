# Self-Check Rules – Commentary
本リポジトリは ChatGPT カスタム設定の**骨格（憲法）**を補完する**逐条解説（法律）**です。  
- 最新骨格: カスタム設定欄（チャット側）  
- 逐条解説: /docs/law/ 以下  
- 運用細則・チェックリスト: /docs/ordinance/  
- 例示: /docs/examples/

## 更新手順（簡易）
1. `docs/law` 等を編集 → PR or 直接コミット  
2. `CHANGELOG.md` に要約を追記  
3. 互換性が変わる変更は `VERSION` を更新（例：v6.0 → v6.1）
